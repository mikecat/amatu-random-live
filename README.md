あまつランダムライブ
====================

[https://mikecat.github.io/amatu-random-live/](https://mikecat.github.io/amatu-random-live/)

## これは何？

[アイドルマスターTOURS](https://bandainamco-am.co.jp/am/vg/idolmaster-tours/) でライブを行う際の楽曲とアイドルをランダムで選択する**非公式**ツールです。  
アイドルマスターTOURS の開発者・運営者・権利者などとは全く関係ありません。  
このツールを用いた、または用いなかったことによる一切の損害について、作者は責任を負いません。

## 使い方

候補となる楽曲とアイドルを設定し、「抽選！」ボタンを押すことで抽選を行います。  
候補となる楽曲に、候補となるアイドルのうちその楽曲で選択できるものが3人以上居るものが無い場合 (すなわち、有効な結果の候補が0件の場合) は、「抽選！」ボタンが押せなくなります。

抽選後、「抽選結果をポスト (ツイート)」ボタンを押すと、X (旧Twitter) で抽選結果を投稿するための画面を開きます。

設定はブラウザ (localStorage) に保存され、次にページを開いた際にデフォルトとして設定されます。
